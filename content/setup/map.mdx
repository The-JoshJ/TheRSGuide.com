### Minimap Quick Tips

<SplitContent>
  <SplitItem width={50}>
  - You can click on the minimap to walk around.
  - You can zoom in and out on the minimap by using the scroll wheel.
  - You can resize your minimap if you're using the modern interface system.
  - You can toggle between a circular minimap and a square minimap and toggle between "Classic" and "Modern" map icons in your "Appearance" settings
  </SplitItem>
  <SplitItem width={50}>
    ![Clock Interface Settings](https://media.thersguide.com/images/map/clockSettings.png)
  </SplitItem>
</SplitContent>

## The World Map

The world map is one of the most helpful tools in your toolkit! It allows you to quickly find, mark, and navigate to any location in Gielinor. You can **zoom, click, drag,** or **search** the map to get a better understanding of your surroundings.

<InteractiveMapMarker
  src="https://media.thersguide.com/images/map/worldMap.png"
  alt="RS World Map"
/>

> *Double clicking the map above will place a marker - give it a try!*

### World Map Quick Tips
- You can zoom in and out on the world map by using the scroll wheel.
- You can Toggle the filters on the left of the screen to show or hide common locations.
- You can select other map areas to the right of the search bar.
- Hovering over skill icons provides useful information about skilling resource locations and training areas.

<SplitContent>

Lodestones can be used to quickly navigate around Gielinor. These can be unlocked by clicking on lodestones you encounter around the game world. Some areas may have quest requirements to access. These cannot be used in combat, but require **no runes** and have **no cooldown**.

## High Contrast Mode

<SplitContent>
  <SplitItem width={50}>
High contrast mode is an accessibility feature which can be toggled on or off at any time. It is especially useful to help distinguish between object types in cluttered areas.

By default, some entity types share the same highlight colour. If you would like more distinction, you can change the highlight parameters of each object type in the *entity highlight* settings.
  </SplitItem>

    ![High Contrast Mode](https://www.thersguide.com/images/map/highcontrast.jpg)

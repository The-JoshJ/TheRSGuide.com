## Choose Option Menu

This section allows you to choose how to attack targets.

### Player Attack Option

Up to your discretion, but generally **right click** is preferred for safety.

### NPC Attack Option

Should be set to **left click**.

### Hide Familiar Options

Should be **enabled**, reducing accidental misclicks.

### Hide Duel Option

Should be **enabled**, reducing accidental misclicks.
